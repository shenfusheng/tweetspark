# TweetSpark MVP 项目文档
**版本：v1.0 | 上线目标：24小时内 | 作者：阿砍 (PM)**

## 🎯 核心理念
**10秒出10条推文灵感**，解决出海创业者"脑子空"的内容创作痛点。

### 目标用户画像
- **独立出海创业者**（Indie Hackers, SaaS创始人）
- **痛点**：每天需要发布有价值的内容，但缺乏灵感和时间
- **付费意愿**：愿意为节省时间支付$5/月
- **验证指标**：**$50月收入**（10个付费用户）

## 📋 MVP范围（极简主义）
> 阿砍哲学：**"代码是负债"**，每多一行代码都是技术债务

### 核心功能（仅3个）
1. **主题输入**：单行文本框 + 热门主题建议标签
2. **AI生成**：调用OpenAI API生成3-5条推文
3. **结果交互**：一键复制 + 简单收藏（localStorage）

### 明确砍掉的功能
- ❌ 用户登录/注册系统（先用localStorage）
- ❌ 内容排期/定时发布
- ❌ 多平台支持（只做Twitter）
- ❌ 数据分析/转化追踪
- ❌ 团队协作功能
- ❌ 内容库/模板管理

## 🏗️ 技术栈选择
| 组件 | 选择 | 理由 |
|------|------|------|
| 前端 | Next.js 14 (App Router) | Vercel零配置部署，一体化架构 |
| 样式 | Tailwind CSS v4 | 快速开发，无需配置文件 |
| 数据库 | Supabase (PostgreSQL) | 免费层够用，自带Auth/实时 |
| AI服务 | OpenAI GPT-4o-mini | 成本最低($0.15/1M tokens) |
| 支付 | Stripe Checkout链接 | 嵌入链接即可收钱 |
| 部署 | Vercel | Git push自动部署 |

### 成功标准
- **正指标**：$50月收入（10个付费用户）

## ⚠️ 风险控制
1. **技术债风险**：坚持500行代码上限
2. **成本风险**：OpenAI API用量监控
3. **法律风险**：内容版权声明
4. **用户风险**：隐私数据保护（localStorage）
